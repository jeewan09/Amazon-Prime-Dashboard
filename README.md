# 📊 Amazon Prime Users Dashboard (Power BI)

## 🔹 Project Overview
This project analyses Amazon Prime users' data to understand customer behaviour, subscription patterns, and device usage.

## 🔹 Key Insights
- Most users prefer Smartphones
- Payment methods are evenly distributed
- Popular genres vary by gender
- Monthly vs Annual subscription trends analysed

## 🔹 Tools Used
- Power BI
- Excel

## 🔹 Files Included
- Power BI Dashboard (.pbix)
- Dashboard PDF
- Images

## 🔹 Business Impact
Helps improve customer targeting, content recommendation, and revenue strategies.
